### Python-сервисы рядом с 1С: PDF → документ + обработка изображений для печатных форм  
**Стек:** Python (FastAPI/Flask), HTTP, 1С

**Сервис 1: Парсинг PDF и заполнение документа 1С**
- Принимает PDF (вложения из 1С)
- Извлекает структурированные данные (табличные/реквизиты)
- Возвращает JSON в 1С, который используется для автозаполнения документа  
- Упрощает работу пользователей и снижает ручной ввод

**Сервис 2: Обработка изображений для печатных форм**
- Принимает несколько изображений (присоединенные файлы из 1С)
- Склеивает/масштабирует в один итоговый файл
- Возвращает результат в 1С для вставки в печатную форму единым блоком  
- Снижает нагрузку на 1С и ускоряет формирование документов
